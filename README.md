# CayenaBot 🏖️

**AI PropTech Platform — CAYENA BEACH CONDOS**

Built with CayenaBot AI Platform — virtual tours, AI chat, render generation.

## Features
- 🧭 Virtual Tour — walk through 13 real project spaces with arrow navigation
- 🖼️ Gallery — all project renders with lightbox
- 🏠 Unit listings — Villas, Townhouses, Apartments with pricing
- 💬 AI Chat — OpenRouter-powered property assistant
- ✨ Render Generator — HuggingFace SDXL photorealistic renders
- 📱 Mobile-first — swipe to navigate, responsive on all devices

## Deploy on Render
1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Static Site
3. Connect your GitHub repo
4. Set Publish Directory to `.`
5. Deploy

## Tech
Single-file HTML app — no build step, no dependencies.
