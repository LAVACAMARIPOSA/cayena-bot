# CayenaBot 🏗️

**AI PropTech Builder — Conversational Content Platform for Real Estate**

CayenaBot is an AI-powered platform that lets you build professional real estate content through conversation. Describe your project and CayenaBot generates renders, virtual tours, galleries, and unit listings automatically.

## Features
- 💬 AI Chat (primary interface) — describe your project and CayenaBot builds it
- 🧭 Virtual Tour — interactive 360° tours generated from project data
- 🖼️ Gallery — AI-generated photorealistic renders
- 🏠 Unit listings — auto-generated from project specs
- ✨ Render Generator — HuggingFace SDXL photorealistic renders
- 🏗️ Planos AI — 3D architectural plan visualization
- 📱 Mobile-first — responsive on all devices

## How it works
1. Chat with CayenaBot on the left panel — provide project name, location, unit types, etc.
2. CayenaBot organizes the info and generates content
3. Preview renders, tours, and listings in the right panel
4. Request adjustments through the chat

## Deploy on Render
1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New → Static Site
3. Connect your GitHub repo
4. Set Publish Directory to `.`
5. Deploy

## Tech
Single-file HTML app — no build step, no dependencies.
